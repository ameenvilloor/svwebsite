AreaMetric_3 (gridDim truePgn approxPgn truePhi approxPhi)

by Nathan Wilson
Calculates the goodness metric from two 3-D objects.
Dangers: This routine currently only works for SparseGrid.
Notes: The equation for the goodness metric is:

goodness = sqrt ( (Aintersect / Atrue) * (Aintersect / Aapprox) )
Argument: gridDim not documented
Argument: truePgn True 3D PolyData object.
Argument: approxPgn Approximate 3D PolyData object.
Argument: truePhi level set scalar function StructuredPts object.
Argument: approxPhi level set scalar function StructuredPts object.
Returns: goodness metric between 0 and 1.

by Ken Wang
This is macro to run a levelset simulation to segment a 2-D contour from a slice of image data.
Argument: img image data (vtkImageData)
Argument: pot potential well data
Argument: initData intialization procedures
Argument: kt curvature value used for vExpDecay function
Argument: klow used by vPotential & vSmooth
Argument: kupp used by vPotential & vSmooth
Argument: serialInit (0/1) indicates whether we are running a set of segmentations in which we want to use the level set function as it was left by the previous item
Argument: gridFactor grid spacing (h) based on min voxel size * gridFactor
Argument: initOnly boolean which determines if you actually run simulation or just set up for it
Argument: sparseFlag whether to use the sparse or dense grid
Argument: isotropicFlag boolean to use a uniformly spaced grid in all dimensions
Argument: expandFlag boolean used to determine if front in
Argument: convFlag boolean determines if area/volume convergence is checked
Argument: timerFlag $core SetTimers -flag $timerFlag
Argument: maxTSFlag boolean determining whether to exist after a fixed number of steps
Argument: maxTS maximum number of time steps to run
Argument: writeIntermediate ** probably output intermediate objects flag **
Argument: runVpot not documented
Argument: runVSmooth boolean to run vSmooth
Argument: runVConst boolean to run VConst
Argument: numVConst number of steps to run VConst
Argument: magGradFactor magnitude of intensity gradient cut-off for vExpDecay
Argument: pFactor potential well tolerance for vPotential
Argument: rens render windows (set to {} for no graphics output)
Argument: objPrefix all objects to be saved created in repository with this prefix
Argument: verbose boolean to output solution results along the way
Argument: stopV stop velocity
Argument: rebuildPhiFreq frequency to rebuild Phi
