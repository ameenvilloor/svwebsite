###Meshing Using MeshSim

	Tab: MeshSim → Mesh
	Type: Parasolid
	File: demo.xmt_txt
	Global Max Edge Size: 0.3  (absolute)
	Click “Run Mesher”
	“Creating mesh can take a long time. Continue anyway?”: "Yes"
	“Use currently selected meshing options?...”: "Yes"

	Click “Write Files”
