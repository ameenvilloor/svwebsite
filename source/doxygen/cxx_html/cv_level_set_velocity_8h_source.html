32 // Class cvLevelSetVelocity is being created in order to encapsulate all of the
33 // details of velocity function evaluation. Previously, methods in
34 // class Grid were used to directly compute velocities. Encapsulation
35 // of velocity parameters will serve to separate the mechanics of Grid
36 // maintenance from the details of particular velocity functions.
