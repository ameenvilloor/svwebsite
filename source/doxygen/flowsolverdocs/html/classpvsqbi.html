Detailed Description

Natural pressure boundary condition can be calculated with p, the pressure, related (in some prescribed manner) to Q, the flow rate, through the same boundary. To do this efficiently requires us to precompute the integral of N_A over the boundary for each node A and store it in a vector of length nshg (a bit wasteful since only the nodes on the boundary will be non zero in this vector but it is probably slower to index it than to multiply and add the extra zeros....check later).
