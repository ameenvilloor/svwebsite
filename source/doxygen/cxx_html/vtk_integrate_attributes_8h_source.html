15 // .NAME vtkIntegrateAttributes - Integrates lines, surfaces and volume.
16 // .SECTION Description
17 // Integrates all point and cell data attributes while computing
18 // length, area or volume. Works for 1D, 2D or 3D. Only one dimensionality
19 // at a time. For volume, this filter ignores all but 3D cells. It
20 // will not compute the volume contained in a closed surface.
21 // The output of this filter is a single point and vertex. The attributes
22 // for this point and cell will contain the integration results
23 // for the corresponding input attributes.
