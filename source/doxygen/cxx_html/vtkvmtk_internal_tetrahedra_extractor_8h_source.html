21  // .NAME vtkvmtkInternalTetrahedraExtractor - Extract internal tetrahedra from a Delaunay tessellation.
22  // .SECTION Description
23  // This class takes in input the Delaunay tessellation of a point set and extracts internal tetrahedra based on outward oriented point normals (to be provided as input point data array). A tetrahedron \f$T_i\f$ is retained if
24  // \f[(x_j - c_i) \cdot n_j \geq 0 \qquad \forall x_j \in T_i \f]
25  // where \f$x_i\f$ are the vertices of \f$T_i\f$, \f$c_i\f$ its circumcenter and \f$n_j\f$ the normals at the vertices.
26  // It is possible to properly handle capped regions (generated with vtkCapPolyData) by activating UseCaps and providing the ids of cap centers.
27  // .SECTION See Also
28  // vtkCapPolyData
