class  cvPolyDataSolid
 

Detailed Description

Class provides implementations of the PolyData solid type.

This is derived from the SolidModel class and provides implementations of functions to be able to import a solid using vtkReaders, extract the boundaries of the solid, and mesh the solid with TetGen.
